## vue有什么性能问题，以及Vue的性能优化有哪些？

（1）代码层面的优化

- 数据层级不要过深，合理的设置响应式数据
- 使用数据时，缓存值的结果，不频繁取值
- v-for合理设置key
- v-show(频繁切换性能高)和v-if的合理使用
- 控制组件的粒度 -> Vue采用组件级别更新
- 采用函数式组件 -> 函数式组价开销低
- 采用异步组件 -> 借助webpack的分包策略
- 事件的销毁
- 图片和路由的懒加载
- 第三方插件的按需引入
- 使用keep-alive来缓存组件
- 虚拟滚动、时间分片等策略
- 服务端渲染 ssr 或者预渲染

（2）Webpack 打包的优化

- 图片压缩
- 减少 ES6 转为 ES5 的冗余代码
- 提取公共代码
- 模板预编译
- 提取组件的 CSS
- 优化 SourceMap
- 编译优化

（3）Web 技术的优化

- 开启 gzip 压缩
- 浏览器缓存
- CDN 的使用
